## Features

### `rpc`

If you want to trigger rpc feature to get data which described in this repo, please 
import `darwinia-bridge-primitives` as below in your `Cargo.toml`

```toml
[dependencies.darwinia-bridge-primitives]
version = "^0"
features = [ "rpc" ]
```
